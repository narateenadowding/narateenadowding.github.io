Introduction to Functions

A relation is a set of ordered pairs. (input,output)

The set consisting of the first components of each ordered pair is called the domain and the second set consisting of the second components of each ordered pair is called the range.

The input values make up the domain, and the output values make up the range.

A function is a relation in which each possible input value leads to exactly one output value. We say “the output is a function of the input.”

ex:

Function Notation f(x) = x. x is the input and f(x) "f of x" is the output.
